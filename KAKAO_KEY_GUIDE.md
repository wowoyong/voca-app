# 카카오 REST API 키 찾는 방법

## 현재 상황
액세스 토큰을 발급받으셨지만, 필요한 것은 **REST API 키**입니다.

---

## REST API 키 확인

1. https://developers.kakao.com 접속
2. 좌측 상단 **내 애플리케이션** 클릭
3. 생성한 앱 선택 (예: Voca App, Wordio 등)
4. 좌측 메뉴 **앱 설정** → **앱 키** 클릭
5. 다음 키들이 표시됩니다:
   - **네이티브 앱 키**
   - **REST API 키** ← 이것이 필요합니다!
   - JavaScript 키
   - Admin 키

## REST API 키 형식

- 길이: 32자
- 형식: 영문자 + 숫자 조합
- 예시: `abc123def456ghi789jkl012mno345pq`

## 액세스 토큰 vs REST API 키

| 구분 | REST API 키 | 액세스 토큰 |
|------|-------------|-------------|
| 위치 | 앱 키 탭 | 발급 후 생성 |
| 용도 | 앱 식별 | 사용자 로그인 세션 |
| 수명 | 영구 | 임시 (2시간~6개월) |
| 형식 | 32자 고정 | 가변 길이 |
| 예시 | abc123...pq | xjOt3OFin... |
