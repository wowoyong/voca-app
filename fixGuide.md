# voca-app UX/UI 수정 가이드

## ✅ 잘 된 점 (강점)
- 홈(빠른 시작) 구조가 직관적이고 바로 학습 행동 유도
- 오늘 학습 플로우가 단순하고 집중 가능
- 하단 탭 구조가 기본 학습 흐름(홈/오늘/퀴즈/통계) 중심

---

## ⚠️ 수정/보완 포인트

### 1) 하단 탭 용어/구조 정리
**문제**
- “오늘”, “플래시카드(빈칸)”, “퀴즈”가 중복 학습으로 느껴짐

**개선**
- 탭을 학습 목적 중심으로 정리
  - 홈 / 학습 / 복습 / 테스트 / 통계
- “오늘/플래시카드/퀴즈”는 **학습 탭 내부 진입형**으로 구성

---

### 2) 홈 화면에서 복습 유도 강화
**문제**
- “복습 대기” 숫자만 있고 행동 유도 약함

**개선**
- 복습 카드 클릭 시 바로 복습 시작
- 복습 배지 강조 + CTA 버튼 추가

---

### 3) 오늘 학습 화면 버튼 의미 혼동
**문제**
- “학습완료” 버튼이 단어 1개 완료인지 전체 완료인지 혼란

**개선**
- 단어 1개 완료 버튼: **“다음”**
- 전체 완료 상태에서만 “오늘 학습 끝내기” 표시

---

### 4) 학습 진행 중 이탈 방지 UX 부족
**문제**
- 진행 중 나가면 유저가 얼마나 학습했는지 시각화 부족

**개선**
- 상단에 **잔여 개수/퍼센트 표시**
- “중단 후 이어하기” 배너 제공

---

### 5) Flashcard/Quiz 진입 기준 불명확
**문제**
- 사용자가 어느 모드를 언제 써야 할지 모름

**개선**
- 각 모드에 **추천 상황 설명** 추가
  - 예: “퀴즈 = 복습 완료 후 테스트용”

---

### 6) Settings 접근성
**문제**
- 홈 상단 아이콘만으로는 설정 위치가 잘 안 보임

**개선**
- 설정 탭 or 하단 메뉴에서 접근 가능하도록 보조 진입 추가

---

## ✅ 우선순위 추천
1) 탭 구조 재정리 (중복 제거)
2) “학습완료” 버튼 명확화
3) 복습 CTA 강화
4) 모드 설명 추가
